### 1. Prepare Your File

| Required | Column Name | Example |
|--------|-----------|--------|
| Yes | `phone` | `+15551234567` |
| Yes | `first_name` | `John` |
| Optional | `last_name` | `Smith` |
| Optional | `email` | `john@email.com` |
| Optional | `tags` | `hot-lead, real-estate` |

> Download **template.csv** from platform

---

### 2. Upload Methods

| Method | Steps |
|------|-------|
| **CSV/Excel** | Drag & drop → Auto-map fields |
| **Google Sheets** | Paste URL → Live sync |
| **Copy-Paste** | Paste 100+ rows → Instant import |

---

### 3. Map Fields

| Your Column | Map To |
|-----------|--------|
| `Phone Number` | `phone` |
| `First` | `first_name` |
| `Lead Source` | `tags` |

Click **Auto-Detect** to speed up.

---

### 4. Client Assignment (Agency)

1. **Upload to Master Account**
2. **Clone List** → `ClientX - Q4 Leads`
3. **Assign to Client Portal**
4. Client sees **only their leads** — no cross-data

---
<Tip>
Use **tags** like `client-abc` to filter in campaigns.
</Tip>
