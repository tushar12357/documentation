### Supported Lead Sources

| Source | Type | Speed |
|--------|------|-------|
| **Website Forms** | Direct Webhook | Instant |
| **Facebook Lead Ads** | Native Sync | Instant |
| **Zapier / Make** | Webhook Push | < 10 sec |
| **Funnels (ClickFunnels, GHL, Leadpages)** | Form → Webhook | Instant |

---

### Setup: Web Form (Most Popular)

1. Go to **Campaigns** → **Create Campaign**
2. Choose **Inbound**
3. Select **Form / Webhook**
4. Map fields:
   - `Phone` → `phone`
   - `First Name` → `first_name`
   - `Email` → `email`
5. Enable **Call Lead Within 60 Seconds**

> Lead submits form → instantly pushed to CloserX → AI calls → books appointment

---

### Client-Specific Routing

1. Create **separate inbound campaign per client**
2. Connect each client’s form/webhook
3. Client sees **only their leads**
4. AI uses **their script + calendar**
