### 1. Connect Cal.com
1. **Agent Editor** → **Tools** → **Calendar**
2. Click **+ Add Cal.com**
3. Paste **Cal.com API Key** (from Settings → API)
4. Select **Event Type**: `30-min Demo`

---

### 2. AI Booking Flow

User: "Tuesday at 2pm"
AI: Checks Cal.com → "Available! Booking now..."
→ Creates event → Sends confirmation


---

### 3. Client Workflow
- Clone agent → Connect to **client’s Cal.com**
- Assign → Client gets **branded booking links**
