## Domain Types Supported

| Type | Example | Works? |
|------|--------|-------|
| Subdomain | `ai.client.com` | Yes |
| Root | `client.com` | No |
| Subfolder | `client.com/ai` | No |

> Only **subdomains** allowed for security.
