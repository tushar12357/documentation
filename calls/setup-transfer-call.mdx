### Transfer Triggers

| Trigger | Action |
|-------|--------|
| Keyword: `"manager"`, `"supervisor"` | Transfer |
| Intent: High-value lead | Auto-transfer |
| 3+ failed responses | Escalate |
| User request | Immediate |

---

### How to Set Up

1. **Agent Editor** → **Tools** → **Call Transfer**
2. Enter **transfer number** (e.g., `+15551234567`)
3. Add **intro message**:
   > “Connecting you to our senior advisor, Mike…”
4. Enable **whisper mode** (AI briefs human before connect)

---

### Agency Workflow

1. **Clone** agent for client
2. Set **client’s support number**
3. Assign via **Client Portal**
4. Client sees: “Your AI → Your Team” seamless handoff
